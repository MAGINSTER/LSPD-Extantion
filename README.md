# Arizona RP Forum Helper

Это базовое расширение Chrome для форума forum.arizona-rp.com.

## Что делает
- добавляет всплывающее окно в браузере;
- открывает форум одним кликом;
- показывает простой помощник на странице форума.

## Установка
1. Откройте Chrome и перейдите по адресу `chrome://extensions`.
2. Включите режим разработчика.
3. Нажмите "Загрузить распакованное расширение".
4. Выберите папку с этим проектом.

## Файлы
- manifest.json — конфигурация расширения;
- popup.html / popup.js / popup.css — popup интерфейс;
- content.js / content.css — интеграция на странице форума.
